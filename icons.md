# Icones em listas ❤

- Dessa forma, será adicionado identidade para mutiplos grupos de tarefa.
- Os icones serão importados do **fonte Awesome**

---

## Passo a passo para adicionar a feature de icones como thumbnaill para listas.

- Não será necessário refatoração em aréas mais acima.

1. Abrir um novo argumento no Objeto `grupo` para aceitar _icons_ uma string.
2. Criar um componente chamado `selectIcon.jsx`.
3. Listar todos os icones que serão adicionado às opções.
4. Adicionar a feature na aréa universal de `Criação e edição de tarefas`

### Considerações finais:

- Verifique com o ChatGPT a respeito de tal modo, para saber se, se aplica aos prinipios `SOLID`.
