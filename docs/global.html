TaskCard(task, forEditionopt, key)

Componente TaskCard ------------------- Renderiza um cartão representando uma tarefa individual. Estrutura: - Exibe um botão de checkbox para marcar a tarefa como concluída. - Exibe o nome da tarefa: - Se `forEdition` for `false`, mostra como link que leva ao detalhe da tarefa. - Se `forEdition` for `true`, mostra como campo de input editável. - Se a tarefa estiver concluída: - O nome recebe estilo "line-through". - O botão é preenchido com um ícone de check e cor de destaque.
