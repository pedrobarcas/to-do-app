Class: TaskDetailView

TaskDetailView(taskDetailVM, taskEditVM, taskRemoveVM, taskDetailComponent)

Classe TaskDetailView --------------------- Responsável por controlar a tela de detalhes de uma tarefa, conectando UI e ViewModels. Estrutura: - Recebe:

Constructor

new TaskDetailView(taskDetailVM, taskEditVM, taskRemoveVM, taskDetailComponent)

Parameters:
Name Type Description
taskDetailVM TaskDetailViewModel Lida com busca e conclusão da tarefa.
taskEditVM EditViewModel Lida com a edição dos dados da tarefa.
taskRemoveVM RemoveViewModel Remove a tarefa do repositório.
taskDetailComponent function Componente que renderiza o detalhe da tarefa. Métodos: - render(root): 1. Obtém a tarefa a partir da `taskDetailVM` usando o `task_id` da URL. 2. Renderiza o componente `taskDetail` dentro do `root`. 3. Seleciona elementos do DOM (inputs e botões). 4. Eventos: - `input`: edita a tarefa em tempo real (nome, anotações, data). - clique no ícone de lixeira: remove a tarefa e redireciona para a home. - clique no botão de checkbox: alterna conclusão da tarefa e recarrega a tela. Exemplo de uso: const view = new TaskDetailView(taskDetailVM, editVM, removeVM, taskDetail); view.render(document.querySelector(".container"));
