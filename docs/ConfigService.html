Class: ConfigService

ConfigService(env)

Serviço responsável por fornecer acesso centralizado a variáveis de configuração da aplicação. Normalmente é usado para encapsular o acesso a variáveis de ambiente (`process.env`), oferecendo uma camada de abstração segura e de fácil manutenção.

Constructor

new ConfigService(env)

Cria uma nova instância do ConfigService.
Parameters:
Name Type Description
env Object.<string, string> Objeto contendo pares chave/valor de variáveis de ambiente.
Since:
  • 1.0.0
Source:
  • domain/services/configService.js, line 16
Example
// Exemplo de uso
const config = new ConfigService(process.env);
const dbUrl = config.get("DATABASE_URL");
console.log(dbUrl);
