Class: CreateTaskViewModel

CreateTaskViewModel(factory, repository)

ViewModel responsável pela criação de tarefas, integrando autenticação do Firebase, a camada de domínio (Factory) e o repositório de persistência. Essa classe estende Observable, permitindo que observadores sejam notificados sempre que uma nova tarefa for criada com sucesso.

Constructor

new CreateTaskViewModel(factory, repository)

Cria uma nova instância do CreateTaskViewModel.
Parameters:
Name Type Description
factory Object Instância responsável pela criação de objetos Task. Deve expor um método `create(name, userId, groupId)`.
repository Object Repositório responsável pela persistência das tarefas. Deve expor um método `save(task)` que retorne uma Promise.
Since:

Example
// Exemplo de uso:
const vm = new CreateTaskViewModel(taskFactory, taskRepository);

vm.subscribe((task) => console.log("Nova tarefa criada:", task));

await vm.create("Estudar TypeScript", "grupo123");

Methods

(async) create(obj, groupId) → {Promise.<Object>}

Cria uma nova tarefa associada ao usuário autenticado e a um grupo específico. Este método aguarda a autenticação do Firebase para obter o `userId` e, em seguida, utiliza a `factory` e o `repository` para construir e persistir a nova tarefa. Após o salvamento bem-sucedido, todos os observadores são notificados via Observable#notify.
Parameters:
Name Type Description
obj string | Object Nome da tarefa ou objeto com dados necessários para criação.
groupId string Identificador do grupo ao qual a tarefa pertence.
