Class: TaskFactory

TaskFactory()

Fábrica responsável pela criação de novas instâncias de Task. -- Essa classe encapsula a lógica de construção de tarefas, garantindo que todas as tarefas sejam criadas com um ID único, data formatada e estrutura consistente.

Methods

(static) create(name, userIdopt, groupIdopt) → {Task}

Cria uma nova instância de Task. -- Essa função gera automaticamente um identificador único (UUID) e define a data de criação no formato brasileiro (dd/mm/yyyy). É ideal para uso em contextos onde tarefas precisam ser instanciadas de forma padronizada, como ao salvar novas tarefas no banco de dados ou ao registrar eventos do usuário.
Parameters:

userId string <optional>
 Identificador opcional do usuário associado à tarefa.

 Identificador opcional do grupo ao qual a tarefa pertence.
