Class: GroupViewModel

GroupViewModel(repository, groupRepository, service)

ViewModel responsável por gerenciar grupos de tarefas ou entidades similares. Essa classe atua como intermediária entre a camada de domínio e a interface, encapsulando operações de leitura, edição e remoção de grupos, além de notificar automaticamente os observadores sobre mudanças relevantes.

Constructor

new GroupViewModel(repository, groupRepository, service)

Cria uma nova instância de GroupViewModel.
Parameters:
Name Type Description
repository Object Repositório principal responsável pelas operações de persistência de grupos. Deve implementar os métodos `find(key)`, `edit(object)` e `remove(object)`.
groupRepository Object Repositório auxiliar responsável por relacionamentos ou índices de grupos. Deve implementar o método `remove(key)`.
service Object Serviço de domínio responsável por aplicar regras de negócio e validações. Deve implementar o método `edit(object, updates)`.
Since:
  • 1.0.0
Source:
  • ui/viewmodels/groupViewModel.js, line 22
Example
const vm = new GroupViewModel(repository, groupRepository, service);

vm.subscribe((event) => console.log("Grupo modificado:", event));

await vm.edit(grupo, { name: "Novo Nome" });
await vm.remove("group-123");
